# uTools-Habitica
对前端完全懵逼的苦逼后端小开发慢慢琢磨的一个针对uTools工具的一个Habitica第三方插件

**目前还处于胚胎阶段，跪求前端指导 ...(ノ ＿ ＿)ノ**

- 现在还不会打包。。可以直接用浏览器打开
